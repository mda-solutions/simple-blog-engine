En el folder que indicamos en settings.ini (en este caso se llama "entries") es donde almacenamos nuestros items, la magia está en nombrarlos de la manera siguiente:

001_Nombre_de_Mi_Post.html

los guiones bajos ( _ ) son importante porque delimitan el orden de nuestros items, en este caso el 001 nos indica la primer posición y el resto el script se encarga de formatearlo y presentarlo como un título legíble :), por ejemplo este mismo item se llama "02_¿Cómo_funciona.html?".

En estos items puedes poner cualquier HTML válido o simplemente texto
